# Introduction au projet MacraSG

MacraSG est un projet de macramé mural documenté comme un projet technique.

L’objectif est double :
- apprendre les bases du macramé (nœuds, matériel, structure),
- montrer comment documenter un projet créatif dans un dépôt GitHub.

Le projet principal consiste en un macramé mural d’environ 40×60 cm, basé sur quelques nœuds essentiels.
