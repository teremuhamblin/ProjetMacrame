# Nœuds pour Débutant
Guide essentiel pour apprendre les bases du macramé.  
Chaque nœud est présenté avec : description, usage, étapes, erreurs fréquentes.

---

## 1. Nœud d’Ancrage (Lark’s Head Knot)
**Usage** — Fixer les cordes sur une barre, un anneau ou un support.

**Étapes**
1. Plier la corde en deux.
2. Passer la boucle derrière la barre.
3. Ramener les deux brins dans la boucle.
4. Serrer proprement.

**Erreurs fréquentes**
- Boucle mal centrée.
- Serrage irrégulier.

---

## 2. Nœud Plat (Square Knot)
**Usage** — Le nœud le plus utilisé en macramé : bracelets, tentures, sacs.

**Étapes**
1. Utiliser 4 brins : 2 centraux, 2 extérieurs.
2. Passer le brin gauche par-dessus les brins centraux.
3. Passer le brin droit par-dessus le brin gauche, puis derrière les brins centraux.
4. Serrer.
5. Répéter en inversant droite/gauche pour compléter le nœud.

**Erreurs fréquentes**
- Ne pas alterner les côtés → torsion involontaire.
- Tension inégale.

---

## 3. Demi-Nœud (Half Knot)
**Usage** — Créer des spirales décoratives.

**Étapes**
1. Même base que le nœud plat.
2. Toujours commencer du même côté.
3. Répéter jusqu’à obtenir une spirale.

**Erreurs fréquentes**
- Changer de côté par erreur → spirale cassée.

---

## 4. Nœud de Surjet (Overhand Knot)
**Usage** — Finitions, perles, arrêts de cordes.

**Étapes**
1. Former une boucle.
2. Passer l’extrémité dans la boucle.
3. Serrer.

---

## 5. Nœud de Montage en Tête d’Alouette Inversé
**Usage** — Fixation plus plate et esthétique.

**Étapes**
1. Plier la corde en deux.
2. Passer la boucle devant la barre.
3. Ramener les brins dans la boucle par l’arrière.
4. Serrer.

---

## Conseils pour Débutants
- Toujours égaliser les longueurs avant de commencer.
- Maintenir une tension constante.
- Travailler sur un support stable.
