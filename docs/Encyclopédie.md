🟫 NŒUDS ESSENTIELS

---

1. Nœud d’alouette

Description
Nœud de fixation utilisé pour attacher les cordes au support (bâton, anneau, branche).

Schéma ASCII

`
Support
 |----------------|
      \    /
       \  /
        ||
        ||
`

Usages
- Début de projet  
- Fixation des cordes  
- Création de franges  

Erreurs fréquentes
- Boucle trop serrée  
- Cordes mal centrées  

---

2. Nœud plat

Description
Le nœud fondamental du macramé.  
Il se réalise avec 4 brins : 2 centraux + 2 extérieurs.

Schéma ASCII

`
Brins : 1 — 2 — 3 — 4

Étape 1 :
1 passe devant 2-3 et derrière 4
4 passe derrière 2-3 et dans la boucle de 1

Étape 2 :
Symétrique
`

Usages
- Bandes régulières  
- Motifs géométriques  
- Zones structurées  

Variantes
- Nœud plat alterné  
- Nœud plat espacé  

---

3. Demi-nœud torsadé

Description
Même principe que le nœud plat, mais toujours du même côté → crée une torsade.

Schéma ASCII

`
1 → passe toujours devant
4 → passe toujours derrière

Résultat :
   / / / / / /
`

Usages
- Torsades décoratives  
- Colonnes verticales  

---

4. Nœud de feston

Description
Permet de créer des lignes diagonales, courbes ou en V.

Schéma ASCII

`
Brin porteur : ------>
Brin actif   :   \ \ \ \
`

Usages
- Motifs en V  
- Lignes diagonales  
- Motifs géométriques complexes

5. Nœud spirale

Description
Variante du demi-nœud torsadé, mais plus longue et régulière.

Schéma ASCII

`
///// spirale /////
`

Usages
- Suspensions  
- Cordons décoratifs  

---

6. Nœud Josephine (nœud de l’amour)

Description
Nœud décoratif formant une boucle entrelacée.

Schéma ASCII

`
  ⟲———⟳
   \   /
    \ /
    / \
   /   \
`

Usages
- Motifs centraux  
- Bijoux macramé  

---

7. Nœud carré inversé

Description
Variante du nœud plat créant une texture alternée.

Schéma ASCII

`
Étape 1 : nœud plat normal
Étape 2 : inversion des brins
Résultat : texture alternée
`

---

8. Nœud de surliure

Description
Permet de maintenir des brins ensemble ou de finir proprement une corde.

Schéma ASCII

`
|||||||
\_/
`
