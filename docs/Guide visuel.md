Guide Visuel du Macramé
Ce guide fournit des schémas ASCII simples pour visualiser les nœuds essentiels.

---

1. Nœud d’Ancrage

`
   Barre
 ─────────
    |
   ( )
  /   \
 /     \
`

---

2. Nœud Plat (Square Knot)

`
Brins : 1 (gauche) - 2 et 3 (centraux) - 4 (droite)

Étape 1 :
1 ----\     /---- 4
        \   /
         \ /
         2 3

Étape 2 :
4 ----/     \---- 1
        /   \
       /     \
      2       3
`

---

3. Demi-Nœud (Spirale)

`
Répéter toujours du même côté :

1 ----\     /---- 4
        \   /
         \ /
         2 3

→ spirale
`

---

4. Nœud de Surjet

`
   O
  /|
 / |
/  |
`

---

5. Nœud de Montage Inversé

`
   Barre
 ─────────
  \     /
   \   /
    \ /
    / \
   /   \
`

---

Astuces Visuelles
- Les brins extérieurs sont les moteurs du mouvement.
- Les brins centraux servent d’axe.
- Les schémas ASCII simplifient la structure mais respectent la logique du nœud.
`
