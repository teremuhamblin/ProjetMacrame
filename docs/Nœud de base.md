# Nœuds de base

## 1. Nœud d’alouette

Utilisé pour fixer les cordes au support.  
Principe : plier la corde en deux, passer la boucle par-dessus le bâton, puis faire passer les deux brins dans la boucle.

## 2. Nœud plat

Nœud central du macramé.  
Il se réalise avec 4 brins (2 centraux, 2 extérieurs) et forme une bande régulière.

## 3. Demi-nœud torsadé

Même principe que le nœud plat, mais toujours du même côté.  
Résultat : une torsade décorative.

## 4. Nœud de feston

Permet de créer des lignes diagonales ou courbes.  
On utilise un brin porteur et un brin qui vient s’enrouler autour.
