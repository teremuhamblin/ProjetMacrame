# Code de Conduite — MacraSG

## 1. Respect

Les contributeurs doivent faire preuve de respect, bienveillance et courtoisie.

## 2. Communication

- Pas d’attaques personnelles  
- Pas de propos discriminatoires  
- Pas de harcèlement

## 3. Collaboration

- Aider les nouveaux contributeurs  
- Fournir des retours constructifs  
- Favoriser un environnement positif

## 4. Application

Les mainteneurs peuvent modérer, fermer ou refuser des contributions ne respectant pas ce code.

Merci de contribuer dans un esprit positif et collaboratif.
