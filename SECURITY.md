# Politique de Sécurité — MacraSG

## Signalement d’un problème

Si vous découvrez une faille ou un comportement problématique :

1. Ne l’annoncez pas publiquement  
2. Contactez le mainteneur du projet  
3. Fournissez une description claire du problème

## Délais

Les problèmes de sécurité seront traités en priorité.

## Portée

MacraSG étant un projet artistique et documentaire, les risques sont faibles, mais toute anomalie est la bienvenue.
